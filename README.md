# Bread Boom 面包热潮 🍞🔥

> 王森冠军联盟 · 桐庐未来湖水街

**Bread Boom 面包热潮** 是王森文创旗下首家全时段面包工坊，位于桐庐未来湖水街 D 座 201。由王森冠军联盟团队操刀，以当地食材为核心、极致性价比为理念、窑炉柴烧为工艺，打造 1100㎡ 的"不追网红、只追风味"的全时段面包西餐空间。
